## Instrucciones de instalación y ejecución.
->Despues de clonar el repositorio https://github.com/CarlosAlbeiro/PARCIAL_ANGULAR

->Realizar la intalacion de las dependencias de node.js mediante el siguiente comando

```bash
npm install
```
->Se demorada unos minutos miestras instala todo, despues de esto podremos correr el poryecto por medio del siguiente comando

```bash
ng serve
```

Este proyecto se desplegara en la siguiente ruta LOCAL `http://localhost:4200/`.
